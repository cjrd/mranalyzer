pd.DataFrame util.combine_multiple_csvs_to_dataframe ( List[str]  csvPaths,
str   sep = "," 
)

Combine multiple csvs into a single dataframe.

This function is useful for combining multiple csvs into a single dataframe, where each csv should have the same number of columns and rows, where if a given row in a particular csv can not be parsed, then that row is not parsed for all csvs, thereby keeping all csvs aligned in the output dataframe.

Parameters
csvPaths(List[str]): List of paths to csvs to combine.
sep(str): Separator to use when parsing csvs.
Returns
(pd.DataFrame): Dataframe containing all data from all csvs.
