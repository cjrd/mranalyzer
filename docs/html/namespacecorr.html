Detailed Description

Perform an exploratory correlation analysis of the input atmospheric data.

This subpackage contains functions for performing an exploratory correlation analysis of the input atmospheric data. The main function for this subpackage is corr(), which can be called from the command line as follows: mra corr; see mra corr --help for more information.
